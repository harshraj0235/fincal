## 📋 Content Creation Checklist (Per Page)

Use this for EVERY page you fix:

### Content Requirements:
- [ ] 800-1000 words minimum (1000-1500 for competitive topics)
- [ ] Clear, unique H1 title
- [ ] Proper H2/H3 heading structure
- [ ] Introduction (150-200 words)
- [ ] Main content sections with subheadings
- [ ] 3+ real-world examples with calculations
- [ ] 5-7 FAQ questions with detailed answers
- [ ] Tips and best practices (5+ items)
- [ ] Common mistakes section (3+ items)
- [ ] Related tools/calculators section (5-7 links)
- [ ] Conclusion paragraph
- [ ] Last updated date visible
- [ ] Author attribution (if applicable)

### Technical SEO:
- [ ] Unique meta title (50-60 characters)
- [ ] Unique meta description (150-160 characters)
- [ ] Canonical URL set correctly
- [ ] WebApplication structured data (for calculators/tools)
- [ ] FAQ structured data (if FAQs present)
- [ ] HowTo structured data (if step-by-step guide)
- [ ] Breadcrumb structured data
- [ ] Open Graph tags
- [ ] Twitter Card tags
- [ ] Alt text for all images
- [ ] Internal links (minimum 5)
- [ ] External authoritative links (2-3)

### User Experience:
- [ ] Fast page load (<2.5s LCP)
- [ ] Mobile-responsive design
- [ ] Clear call-to-action
- [ ] Working calculator/tool functionality
- [ ] Save/share/print options
- [ ] No intrusive popups
- [ ] Accessible (keyboard navigation, screen readers)

### Trust Signals:
- [ ] Disclaimer (for financial advice)
- [ ] Privacy policy link
- [ ] Terms of service link
- [ ] Contact information available
- [ ] SSL certificate (HTTPS)
