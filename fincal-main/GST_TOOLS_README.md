# 🏦 MoneyCal - Indian Financial Platform

## 🎯 Vision: India's Complete Financial Education & Tools Platform

MoneyCal is building India's most comprehensive financial platform - inspired by Investopedia but tailored for Indian markets, regulations, and languages.

---

## ✅ What's Built (Current Status)

### 🧮 GST Tools Suite (5 Tools Live)

1. **GSTR-1 Deadline Calculator**
   - Monthly & quarterly filing support
   - Penalty calculation (₹200/day, max ₹5,000)
   - Interest calculation (18% p.a.)
   - Real-time deadline tracking
   - 🔗 URL: `/tools/gstr-1-deadline-calculator`

2. **GST Amount Calculator**
   - CGST, SGST, IGST breakdown
   - Intra-state & inter-state support
   - Discount support (% & fixed)
   - Calculation history
   - 🔗 URL: `/tools/gst-amount-calculator`

3. **Reverse GST Calculator**
   - Extract base amount from GST-inclusive price
   - Separate GST components
   - Verification feature
   - Quick examples
   - 🔗 URL: `/tools/reverse-gst-calculator`

4. **GST Slab Calculator**
   - 100+ products database
   - HSN code search
   - Category filtering
   - All 5 GST rates (0%, 5%, 12%, 18%, 28%)
   - 🔗 URL: `/tools/gst-slab-calculator`

5. **GSTR-3B Deadline Calculator**
   - Monthly & QRMP scheme support
   - Turnover-based eligibility
   - PMT-06 challan support
   - Auto due date calculation
   - 🔗 URL: `/tools/gstr-3b-deadline-calculator`
